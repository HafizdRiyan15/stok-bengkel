# Stok Bengkel

Website sederhana untuk mengelola stok sparepart bengkel motor.

## Fitur
- Tambah barang
- Update stok
- Hapus barang
- Penyimpanan data menggunakan localStorage

## Tujuan
Membantu mencatat dan memantau stok barang di bengkel agar lebih rapi dan efisien.

## Teknologi
- HTML
- CSS
- JavaScript

Masih dalam pengembangan (versi 1)
