❓ Panduan Penggunaan

Stok Sparepart Bengkel

📦 Stok Barang

  • Tambah barang baru dengan isi nama, stok awal, harga beli, harga jual
  • Pilih kategori masuk: Stok Awal, Pembelian, atau Lainnya
  • Tombol + Masuk untuk tambah stok barang yang sudah ada
  • Tombol − Keluar untuk kurangi stok secara manual
  • Tombol Ubah untuk edit nama, stok, atau harga
  • Harga beli otomatis dihitung rata-rata saat ada pemasukan baru

⚙️ Kelola Jasa

  • Tambah daftar jasa bengkel beserta tarifnya (contoh: Ganti Oli, Tune Up)
  • Jasa tidak punya stok, hanya nama dan tarif
  • Jasa bisa dipilih saat proses transaksi di Kasir

🛒 Kasir

  • Pilih sparepart dan jasa yang dibutuhkan pelanggan
  • Gunakan tombol +/− untuk atur jumlah di keranjang
  • Isi nama pelanggan (opsional)
  • Klik Proses Transaksi untuk selesaikan dan cetak nota
  • Stok otomatis berkurang setelah transaksi diproses

📊 Laporan

  • Lihat rekap pemasukan, pengeluaran, pendapatan jasa, dan keuntungan kotor
  • Filter berdasarkan periode: Hari ini, 7 Hari, Bulan ini, atau Semua

💾 Backup Data

  • Export — download semua data sebagai file backup (.json)
  • Import — pulihkan data dari file backup
  • Lakukan export rutin agar data aman jika browser di-reset
  • Untuk pindah ke HP: export di laptop → kirim file → import di browser HP

⚠️ Penting

Data tersimpan di browser perangkat ini. Gunakan Export secara rutin sebagai cadangan agar data tidak hilang.
